# Open-AutoGLM-Android

项目基于@zai-org/Open-AutoGLM进行二次开发，实现完全手机端自动化

# 1.获取API Key

在https://bigmodel.cn/ 获取API key，需要绑定阿里云并实名认证

# 2.安装软件并配置

下载安装好软件后，会弹出申请无障碍权限，在系统里为本应用开启无障碍即可

# 3.配置API key

在软件中填入key

输入任务描述

在数字键盘中隐藏输入法

点击开始任务
